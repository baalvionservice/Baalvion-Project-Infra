# Security Policy

The security of the Baalvion platform and its user data is our top priority. We are committed to responsible security practices and appreciate the help of the community in keeping our platform safe.

## Supported Versions

We provide security updates for the latest major version of the Baalvion Jobs Portal. Please ensure you are using the most up-to-date version of the application.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, we encourage you to report it to us privately to protect our users' information. Please do not disclose the vulnerability publicly until it has been addressed.

**How to Report:**

1.  **Email us directly** at `security@baalvion.com`. Please use the subject line "Security Vulnerability Report".
2.  Provide a **detailed description** of the vulnerability, including:
    *   The steps to reproduce it.
    *   The potential impact of the vulnerability.
    *   Any suggested mitigation, if you have one.
3.  Include any relevant logs, screenshots, or code snippets that can help us understand the issue.

We will make every effort to acknowledge your report within 48 hours and will keep you informed of our progress as we investigate and resolve the issue.

We greatly appreciate the efforts of security researchers and developers who share their expertise to help us make Baalvion more secure.
